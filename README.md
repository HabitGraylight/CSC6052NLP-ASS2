# CSC6052NLP-ASS2

This repository contains the initial open-source release of key scripts for:

- SFT data processing
- SFT training
- model download helper
- evaluation on GSM8K
- GRPO training launch script (verl)

## Included Files

- `data/data_process/sft_data_proc.py`
- `ft/train.py`
- `download.py`
- `eval/evaluation.py`
- `verl/run_qwen3_0.6b_grpo_gsm8k.sh`

More files and documentation will be added in later updates.
